# Security Policy

## Supported Versions

We release updates for the current major version. Security fixes may be backported to the latest minor of the previous major when applicable.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please do **not** report security vulnerabilities in public issues.

To report a vulnerability:

1. Open a [new security advisory](https://github.com/general-alexson/cursor-project-memory/security/advisories/new) on GitHub, or
2. Email the maintainers (see repository contacts) with a description of the issue and steps to reproduce.

We will acknowledge your report and work with you on a fix and disclosure timeline.
