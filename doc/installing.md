# Installing

## From the marketplace (Open VSX)

1. Open **Extensions** (Ctrl+Shift+X / Cmd+Shift+X).
2. Search for **Project Memory**.
3. Click **Install**.

Or install from [Open VSX](https://open-vsx.org/extension/cursor-project-memory/cursor-project-memory).

## From a VSIX file

1. Download the `.vsix` from the [Releases](https://github.com/general-alexson/cursor-project-memory/releases) page.
2. In VS Code or Cursor: **Extensions** → **...** (More Actions) → **Install from VSIX...**.
3. Select the downloaded file.
